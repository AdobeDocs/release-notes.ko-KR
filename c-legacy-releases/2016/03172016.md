# 릴리스 노트 - 2016년 3월

Adobe Marketing Cloud의 새로운 기능과 수정된 내용에 대해 알아보십시오.

>[!NOTE]
>
>[Adobe Priority Product Update](https://www.adobe.com/subscription/priority-product-update.html)에 가입하면 릴리스 노트를 월별 제품 업데이트 1주일 전에 받아 볼 수 있습니다. Priority Product Update의 릴리스 정보는 릴리스 날짜보다 1주일 전에 제공되며 변경될 수 있습니다. 업데이트의 릴리스 시간을 다시 확인하십시오.

## Experience Cloud

Experience Cloud 및 핵심 서비스에 대한 릴리스 노트입니다.

### Adobe Mobile Services(2016년 3월 9일) {#mobile}

iOS SDK 버전 4.8.6(2016년 3월 9일)에는 다음 변경 사항이 포함됩니다.

| 기능 | 설명 |
|--- |--- |
| 앱 충돌 추적 | iOS SDK 버전 4.8.6에는 잘못된 충돌이 보고되지 못하게 하는 중대한 변경 사항이 포함되어 있습니다. 버전 4.8.6으로 업데이트할 것을 강력히 권장합니다. |

제품 설명서가 필요하면 [Adobe Mobile Services](https://docs.adobe.com/content/help/en/mobile-services/using/home.html)를 확인하십시오. 이전 릴리스의 릴리스 노트를 보려면 왼쪽 창에서 *이전 릴리스 노트*&#x200B;를 확장하십시오.

### Marketing Cloud ID 서비스 {#section_5BA06A92D6E54715B025D86C192B2A8B}

2016년 3월 17일

1.5.4버전에는 다음과 같은 변경 사항이 포함됩니다.

| 기능 | 설명 |
|--- |--- |
| 옵트아웃 지원 | Marketing Cloud ID 서비스가 방문자의 옵트아웃 요청을 지원합니다. (AAM-21725) |
| ID 동기화 간격 변경 | 이제 Marketing Cloud ID 서비스에서 데이터 수집 서버를 호출할 때마다 ID 동기화 호출을 실행합니다. 이전에는 ID 서비스에서 Marketing Cloud ID를 받을 때 요청을 한번만 실행했습니다. (AAM-22934) |

**설명서**

수정된 [!DNL  Analytics] 고객용 ID 서비스 구현 절차. [Marketing Cloud ID Service 및 Analytics 설정하기](https://marketing.adobe.com/resources/help/en_US/mcvid/?f=mcvid-setup-analytics.html)를 참조하십시오.
