# Adobe Experience Cloud 릴리스 정보 - 2021년 6월

## ![아이콘](/assets/ec_appicon_24.png) Experience Cloud 중앙 UI 구성 요소 {#ecloud}

Experience Cloud 중앙 인터페이스 구성 요소에는 자체 도움말, 검색 및 사용자 계정 환경 설정과 같은 통합 제품 헤더에서 액세스할 수 있는 업데이트가 포함되어 있습니다. 인물, 장소(위치) 및 제품 관리에 대한 업데이트는 여기에서 찾을 수 있습니다.

| 기능 | 날짜 | 설명 |
| ------- | ------- | ------- |
| Adobe Federated ID용 SSO(Single Sign-On) 지원 | 2021년 6월 17일 | Federated ID를 사용하는 경우, 이메일 주소 또는 암호를 입력할 필요 없이 Experience Cloud에 로그인할 수 있습니다. 이 기능을 사용하려면 **#/sso:@domain**&#x200B;을 Experience Cloud URL에 추가하십시오. <br><br>예를 들어 소유하고 있는 도메인이 **adobecustomer.com**&#x200B;이고 Adobe Analytics에 로그인하려고 한다고 가정하겠습니다. URL은 **https://experience.adobe.com/#/sso:@adobecustomer.com/analytics**&#x200B;가 될 것입니다. |
| Experience League 검색 | 2021년 6월 1일 | Experience League 설명서 검색이 개선되었습니다. [Experience League](https://experienceleague.adobe.com/docs/)로 이동하고 **[!UICONTROL 검색]** 필드를 사용하여 튜토리얼, 설명서, 과정 등을 찾을 수 있습니다. |
