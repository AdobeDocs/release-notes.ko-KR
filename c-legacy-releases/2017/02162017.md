**수정 사항, 개선 사항 및 중단 사항**

* 사용자 그룹에 RBAC(역할 기반 액세스 제어)를 할당한 일부 고객이 일반 보고서에서 세그먼트를 선택할 수 없는 버그를 수정했습니다. (AAM-28195)
* 데이터 소스 API 대신 세그먼트 API를 잘못 호출하여 *`MergeRule`* 이름을 가져오는 인터페이스 버그를 수정했습니다. (AAM-27473)
* 숫자를 더 잘 설명하는 데 도움이 되도록 [세그먼트 요약] 페이지에서 실시간 및 총 세그먼트 모집단 수에 도구 설명을 추가했습니다. (AAM-28077)
* 특정 고객의 경우 Audience Marketplace의 새 데이터 피드 목록에서 데이터 소스가 누락되었던 버그를 수정했습니다. (AAM-28280)
* 고객에 대한 알고리즘 모델에 삭제된 데이터 소스가 여전히 표시되어 모델이 제대로 실행되지 않던 버그를 수정했습니다. (AAM-28179)
* 그래프에 일반 24-36시간이 아닌 최근 3일의 정보가 표시되는 세그먼트 요약 페이지 및 트레이트 요약 페이지의 버그가 수정되었습니다. (AAM-27988)
* 폴더 API의 응답 시간에 성능이 개선되었습니다. (AAM-28202)
* 지정 가능한 대상 용어를 설명하도록 벤 다이어그램을 업데이트했습니다. See [Addressable Audience documentation](https://marketing.adobe.com/resources/help/en_US/aam/addressable-audiences.html). (AAM-27559)
* 아웃바운드 데이터 파일에서 새로운 &lt;DPUUIDS> 콘텐츠 매크로를 추가했습니다. 아웃바운드 [템플릿 매크로](https://marketing.adobe.com/resources/help/en_US/aam/outbound-template-macros.html) 및 [아웃바운드 매크로 예를 참조하십시오](https://marketing.adobe.com/resources/help/en_US/aam/outbound-macro-examples.html). (AAM-28000)
* 일부 고객이 세그먼트를 편집할 수 없는 버그를 수정했습니다. (AAM-28412)
