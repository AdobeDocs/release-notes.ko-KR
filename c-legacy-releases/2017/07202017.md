**버그 수정 및 기타 변경**

* ID 서비스와 Target이 Internet Explorer에서 함께 작동하지 못하게 하는 VisitorAPI.js v2.2의 버그를 수정했습니다. (MCID-456)
* ID 서비스에서 대상 게시 iFrame에 데이터를 보내는 방법을 개선하는 데 도움이 되도록 코드를 수정했습니다. 이렇게 하면 CPU 사용량이 줄어듭니다. (MCID-418)
