**Analysis Workspace 공간 수정 사항**

* 페이지 체류 시간에 영향을 주는 문제가 해결되었습니다. 작업 공간 보고서는 체류 시간 버킷을 계산할 때 페이지 이름을 더 이상 사용하지 않으므로 세분화되고 버킷 히트가 카운트됩니다. (AN-140479)
* 작업 공간 성능을 개선하기 위한 노력의 일환으로 라인 시각화 성능 문제가 수정되었습니다. (AN-174878)
* 다운로드한 .csv 파일에서 UTF-8 인코딩이 없는 문제가 해결되었습니다. (AN-178393)
* 작업 공간 프로젝트 성능이 저하되는 문제가 해결되었습니다. (AN-177710)
* Y축의 세부기간에 작은 범위의 라인 시각화 표시 문제를 해결했습니다. (AN-176467)

**기타 Analytics 수정 사항**

* Audience Analytics:Audience Manager(AAM)에서 대상자 이름이 변경된 후 발생하는 문제가 수정되었습니다. 업데이트된 이름이 Audience Analytics에 반영되지 않았습니다. (AN-176237)
* 사용자가 AAM에서 Analytics 세그먼트를 저장할 수 없는 문제를 수정했습니다. 이 문제는 대문자와 소문자를 혼합한 기존 AAM 폴더에 의해 발생했습니다. 이제 모든 폴더를 동기화할 수 있도록 대/소문자를 구분하지 않는 폴더로 취급합니다. (AN-177934)
* 사용자가 Experience Cloud을 통해 Analytics에 로그인한 다음 세션 시간이 초과되는 문제를 해결했습니다. 세션을 다시 시작하면 사용자가 잘못된 URL로 리디렉션되었습니다. (AN-176812)
* Data Warehouse 요청의 표준 시간대 오프셋 문제를 수정했습니다. (AN-177585)

## Target Standard/Premium 19.5.1 {#target}

이 릴리스에는 다음과 같은 기능, 변경 사항 및 개선 사항이 포함되었습니다.

(괄호로 묶인 문제 번호는 내부 Adobe용입니다.)

### 기능 업데이트

| 기능/개선 사항 | 설명 |
| --- | --- |
| SPA VEC(Single Page App Visual Experience Composer) | SPA VEC에는 작업을 보다 빠르고 효율적으로 수행할 수 있도록 다음과 같은 개선 사항이 포함되어 있습니다.<ul><li>이제 VEC에서 웹 사이트 로딩을 취소하여 활동 편집의 차단을 해제할 수 있습니다. 이 개선 사항은 예를 들어 활동에 대해 약간 편집하거나, 설정을 검토하거나, 사용자 지정 코드를 추가하려는 경우 사이트가 로드될 때까지 기다리지 않으려는 경우 유용합니다. (TGT-33872)</li><li>페이지가 VEC에 로드되기 전에 또는 페이지가 완전히 로드되지 않은 경우에도 여러 작업을 수행할 수 있습니다(예: 사용자 지정 코드가 더 이상 작동하지 않음). 사이트가 로드되기 전에 편집할 수 없는 작업은 Target UI에서 비활성화됩니다. (TGT-33851 &amp; TGT-34149)</li></ul> |
| Automated Personalization(AP) 및 자동 Target 활동 | AP 또는 자동 Target 활동을 만들 때 제어로 사용할 경험을 선택할 수 있습니다. 이 기능을 사용하면 활동에 구성된 트래픽 할당 비율을 기준으로 전체 제어 트래픽을 특정 경험으로 라우팅할 수 있습니다. 그러면 제어 경험에 대해 개인화된 게재의 성과를 평가할 수 있습니다. (TGT-26572) |
| 추천 항목 | 최근에 본 항목 논리 생성 중에 이전에 구입한 항목 추천 전환을 사용할 수 있습니다. (TGT-34030) |

### 개선 사항, 수정 사항 및 변경 사항

* 도구 모음 아이콘은 VEC 내에서 페이지 로드를 취소한 후에 적절하게 표시됩니다. 페이지가 완전히 로드될 때까지 특정 작업을 수행할 수 없는 경우 연결된 도구 모음 아이콘이 비활성화됩니다. (TGT-33811)
* 이제 플랫 폴더 계층 구조를 탐색하지 않고 자산 선택기에서 오퍼 폴더를 보다 쉽게 나열하고 탐색할 수 있습니다. (TGT-33725)
