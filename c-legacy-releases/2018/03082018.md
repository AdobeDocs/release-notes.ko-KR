### [!DNL Experience Cloud] ID 서비스 {#mcvid}

Experience Cloud ID 서비스에 대한 개선 사항 및 수정 사항입니다.

* 하위 도메인에서 *`resetBeforeVersion`* 함수가 작동하지 않던 버그를 수정했습니다. (CORE-11390)
* [!DNL  visitor.cookieDomain]에 선행 점이 있을 때 Experience Cloud ID 서비스가 선행 점이 없는 도메인에 대해 AMCV 쿠키를 잘못 설정하던 버그를 수정했습니다. For example, it would set an [AMCV cookie](https://marketing.adobe.com/resources/help/en_US/whitepapers/cookies/cookies_mc.html) on [!DNL  sampledomain.com] in addition to [!DNL  .sampledomain.com]. (CORE-10634)

[Experience Cloud ID Service](https://marketing.adobe.com/resources/help/en_US/mcvid/) 제품 설명서를 참조하십시오.
