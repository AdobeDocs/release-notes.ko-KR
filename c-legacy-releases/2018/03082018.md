### [!DNL Experience Cloud] ID 서비스 {#mcvid}

Experience Cloud ID 서비스에 대한 개선 사항 및 수정 사항입니다.

* 하위 도메인에서 함수가 *`resetBeforeVersion`* 작동하지 않던 버그를 수정했습니다. (CORE-11390)
* [!DNL  visitor.cookieDomain]에 선행 점이 있을 때 Experience Cloud ID 서비스가 선행 점이 없는 도메인에 대해 AMCV 쿠키를 잘못 설정하던 버그를 수정했습니다. 예를 들어, AMCV 쿠키는 [AMCV](https://marketing.adobe.com/resources/help/en_US/whitepapers/cookies/cookies_mc.html) 쿠키 [!DNL  sampledomain.com] 와 함께 [!DNL  .sampledomain.com]설정됩니다. (CORE-10634)

[Experience Cloud ID Service](https://docs.adobe.com/content/help/ko-KR/id-service/using/home.html) 제품 설명서를 참조하십시오.

* 페이지에서 다른 선택을 할 때마다 라디오 단추가 **[!UICONTROL 고객 ID]**&#x200B;로 재설정되던 데이터 소스 설정 버그를 수정했습니다. 이 수정에는 데이터 소스 설정에 대한 일부 오류 메시지 불일치가 포함됩니다. (AAM-34754)
