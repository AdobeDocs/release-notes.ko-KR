## [!DNL Adobe System Status] {#status}

[!DNL Adobe System Status]는 Adobe 제품 및 서비스에 대한 자세한 정보, 상태 업데이트 및 이메일 알림을 제공합니다. 중단 및 유지 관리 이벤트에 대한 알림을 받습니다. [status.adobe.com](https://status.adobe.com/)에서 관련 정보를 확인하십시오.

+++릴리스 정보

릴리스 일자: **2025년 7월 16일 목요일**

**새로운 기능**

* Virtual Assistant 일반 가용성
* 이벤트 ID 검색
* Slack 알림 설정 업데이트
* 버그 수정 및 개선 사항

| 기능 | 설명 |
| ------- | ------- |
| **Virtual Assistant 일반 가용성** | <ul><li>**[!UICONTROL Virtual Assistant]**&#x200B;을 통해 웹 또는 모바일의 Adobe 상태 고객이 홈 페이지를 떠나지 않고 답변을 찾고 빠른 작업을 수행할 수 있습니다. 한 곳에서 도움을 받고 신속하게 작업을 완료할 수 있습니다. </li><li>이제 모든 Adobe Status 고객이 Virtual Assistant를 사용할 수 있습니다. [status.adobe.com](https://status.adobe.com/)에 로그인하고 **[!UICONTROL 도우미에게 문의]** 위젯과 상호 작용하면 액세스할 수 있습니다.</li></ul> |
| **이벤트 ID 검색** | <ul><li>고객은 이제 Virtual Assistant뿐만 아니라 제품 및 클라우드 페이지에서 이벤트 ID로 검색할 수 있습니다.</li></ul> |
