### Adobe Analytics의 수정 사항 {#aa-fixes}

* Adobe Analytics 2.0 API 게이트웨이 시간 초과를 300초(5분)로 늘릴 때 발생하는 문제를 해결했습니다. (AN-232335)
* Workspace, API 2.0 및 Adobe Report Builder 보고와 관련된 성능 문제 해결(AN-245644, AN-244504, AN-243060)
* Analytics [!UICONTROL 데이터 피드]에서 **추가**&#x200B;를 클릭할 때 오류가 발생하는 문제를 해결했습니다. (AN-243171)
* 데이터를 분류하지 않는 분류와 관련된 문제를 해결했습니다. (AN-243823, AN-247049, AN-244114)
* 예약된 프로젝트 관련 문제 해결: 고객은 소유한 프로젝트만 볼 수 있고 일부 예약된 프로젝트는 볼 수 없음(AN-246955)
* [!UICONTROL Workspace]의 A4T 패널에서 프로젝트가 열리지 않는 문제를 해결했습니다. (AN-246881)
* [!UICONTROL Workspace]에서 A4T [!UICONTROL 산정된 지표]와 관련된 오류가 발생하는 문제를 해결했습니다. (AN-247082)
* Data Warehouse API 요청에서 데이터를 반환하지 않는 문제를 해결했습니다. (AN-236931)
* 가상 보고서 세트에 대한 액세스와 관련된 문제를 해결했습니다(상위 보고서 세트에 대한 액세스에 대해서만 해당). (AN-247042)
* [!UICONTROL Ad Hoc Analysis]에서 [!UICONTROL Workspace]로 프로젝트를 변환할 때 오류가 발생하는 문제를 해결했습니다. (AN-221215)
* [!UICONTROL Workspace 프로젝트 관리자]에서 필터링된 프로젝트 수가 잘못 표시되는 문제를 해결했습니다. (AN-244934)

         * 콘텐츠 조각 모델에 태그를 지정하고 태그 또는 경로별 정책이 있는 폴더에서 콘텐츠 조각을 만들 수 있습니다.
         * 조각이 기반으로 하는 모델의 게시 작업 및 표시를 포함하여 콘텐츠 조각 편집기의 사용성이 향상되었습니다.
